# 🎯 AI-Based Career Path Recommender

An AI-powered web app that recommends the best career paths based on your **skills, interests, education, and experience** using Machine Learning.

---

## 🚀 Features

- AI-driven personalized career recommendations
- Top 3 career matches with descriptions
- Clean and interactive Streamlit web interface
- Powered by Random Forest + Cosine Similarity

---

## 🛠️ Tech Stack

- **Python**
- **Streamlit** – Web UI
- **Scikit-learn** – Machine Learning (Random Forest, CountVectorizer, LabelEncoder, Cosine Similarity)
- **Pandas** – Data handling
- **NumPy** – Numerical operations

---

## ⚙️ How to Run

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/ai-career-recommender.git
cd ai-career-recommender
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the app
```bash
streamlit run app.py
```

### 4. Open in browser
Go to 👉 `http://localhost:8501`

---

## 📸 How to Use

1. Enter your **skills** (e.g., python, data analysis)
2. Select your **education level**
3. Enter your **interests** (e.g., research, analytics)
4. Select your **experience level**
5. Click **"Recommend Careers"** and get your top 3 matches!
